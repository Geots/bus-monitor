# Real-Time Bus Fleet Monitoring System including:

- Live Admin fleet monitoring Web App with information on each Bus and Alerts in case of emergency.
- Mobile App with route navigation and number of Students onboard for the Driver.
- Mobile App with Real-Time Bus location and route as well as Speed and Alerts for the Parent.
